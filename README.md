# ring_area_calculator

## Features
- Takes user input for outer and inner radius
- Uses a reusable function to calculate circle area
- Uses constant value of π
- Outputs the final ring area

## How It Works
1. User enters outer radius
2. User enters inner radius
3. Program calculates:
   - Area of outer circle
   - Area of inner circle
4. Subtracts both values to get ring area
5. Displays result
